Descrição curta 

CLI de controle de estoque em Python (POO) com CRUD completo, persistência em JSON e validação de entradas do usuário.

✨ Funcionalidades


Cadastrar produtos — nome, quantidade e preço, com validação de dados
Listar produtos — exibe todos os produtos cadastrados, incluindo valor total calculado
Remover produtos — busca por nome e remove do estoque
Repor estoque — adiciona quantidade a um produto já existente
Persistência em arquivo — salva e carrega o estoque automaticamente em formato JSON, sem perder dados entre execuções
Validação de entradas — tratamento de erros para impedir que valores inválidos (texto no lugar de número, nomes vazios) quebrem o programa


🛠️ Tecnologias e conceitos aplicados


Python 3
Programação Orientada a Objetos (classes, atributos, métodos)
Métodos (__init__, __str__)
Manipulação de arquivos (módulo json)
Tratamento de exceções (try/except)
Estruturas de repetição e controle de fluxo (while, for, if/elif/else)
